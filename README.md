# afizan Automation Framework

## Overview
This project demonstrates a scalable QA automation framework using Playwright and TypeScript.

## Features
- UI Automation
- API Automation
- Page Object Model
- GitHub Actions CI/CD

## Tech Stack
- Playwright
- TypeScript
- Node.js

## Installation

```bash
npm install
```

## Run Tests

```bash
npx playwright test
```

## Run Specific Test

```bash
npx playwright test tests/login.spec.ts
```

## Generate HTML Report

```bash
npx playwright show-report
```
